## Environment Vaiables (envvars)

| Windows | Description |
|----| ---|
| %CD% | Current directory |
%DATE% |Current date |
%OS% | Windows |
%ProgramFiles% | C:\Program Files |
%ProgramFiles(x86)% | C:\Program Files (x86) |
%TIME% | Current time |
%USERPROFILE% | C:\Users\{username}
%SYSTEMDRIVE% | C:\ |
%SYSTEMROOT% | C:\Windows |
